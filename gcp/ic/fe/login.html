Data Discovery and Access Platform

Welcome to the GA4GH Demo of federated identity and claims-based access management.

The Data Discovery and Access Platform provides researchers access to data provided by data hosts. Researchers can bring their own identities from various identity providers and aggregate GA4GH Researcher Identity Claims from these accounts to meet access criteria for these research datasets.

This Playground Edition of the platform can be used for testing of claims by allowing users to log in using Test Persona accounts. These test persona accounts can be modified to assert different Identity Claims and then be used to test data access against policies that check those claims. Only for use with synthetic and public datasets.
